🏗️ 设计方案 · 第一部分：系统架构

智言 AI 智能客服系统 — 整体架构图

分层架构

🌐 前端层 (Vue3 + Vite)
登录/注册页 | 会话列表 | 聊天窗口 | 模板管理
⬇️ HTTP/SSE ⬆️ JSON
🖥️ API 层 (Flask)
JWT中间件 | 路由分发 | 请求校验 | SSE推送
⬇️ ⬆️
🤖 AI 服务层 (LangChain)
LLM封装 | Agent编排 | Tools注册 | Memory管理 | Prompt管理
⬇️ ⬆️
🗄️ 数据层
MySQL（用户/会话/消息/模板） | Redis（可选：缓存+会话）

核心实体关系

User 1 ── N Conversation
Conversation 1 ── N Message
Conversation N ── 1 PromptTemplate
PromptTemplate 1 ── N Tool（多对多关联）

数据流 — 一次AI对话请求

① 用户输入消息 → Vue3 前端
② POST /api/chat (含 conversation_id, message, template_id)
③ Flask 校验 JWT → 加载会话历史(从MySQL) → 构建 Memory
④ LangChain Agent 处理：
  → PromptTemplate 渲染系统提示词
  → Memory 注入历史上下文
  → Agent 判断是否调用 Tool
  → 调用工具（模拟订单/库存查询）
  → LLM 生成最终回复
⑤ SSE 流式推送到前端
⑥ 消息异步写入 MySQL
