前端技术要点

  • 状态管理： Pinia（Vue3 官方推荐）
    用户信息 / 会话列表 / 当前消息 / 工具调用状态
  • HTTP 请求： Axios + 拦截器统一处理 JWT
    401 自动跳转登录页
  • SSE 处理： EventSource 或 fetch + ReadableStream
    推荐 fetch 方案，更灵活控制
  • UI 组件： 不引入重型 UI 框架
    自己写简洁的 CSS，面试时能解释每个样式
