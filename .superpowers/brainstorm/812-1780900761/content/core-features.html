🎯 核心功能清单（B方案）

按优先级分三层，确保核心做扎实

🥇 P0 — 必须做（面试必问）

1. 用户登录注册
用户名 + 密码（hash），JWT 鉴权。展示你懂用户体系

2. 多会话管理
新建/切换/删除会话，每个会话独立上下文。展示你懂对话场景设计

3. AI 对话 + 流式输出
用户发消息 → SSE 流式返回 → 前端打字机效果。展示你懂 Streaming

4. 多轮对话记忆
ChatMessageHistory + 滑动窗口，AI 记得前文。展示你懂 Memory

5. Agent + 工具调用
AI 自主判断调工具：查订单、查库存、退换货规则。展示你懂 Agent

🥈 P1 — 应该做（加分项）

6. 提示词模板切换
预设多套角色模板（客服默认/售后专员/技术顾问），用户可切换

7. 工具调用可视化
前端展示 AI 正在调用的工具和参数（"正在查询订单 OR2024001..."）

🥉 P2 — 有时间再做（锦上添花）

8. Docker 部署 — docker-compose 一键启动
9. 单元测试 — API 测试 + Agent 测试
10. RAG 知识库 — 预留接口，时间充裕再加
